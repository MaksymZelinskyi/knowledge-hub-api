# 🚀 Knowledge Hub API

A forum-api with a jwt-authentication

## 🛠️ Features

- ✅ Role-based User authentication with Spring Security
- ✨ View articles
- 📝 Create, edit, and delete articles
- ❤️ Comment articles
- 🔎 Search for articles by tags


## 🧰 Tech Stack

- **Java 17**
- **Spring Boot**
- **PostgreSQL**
- **Hibernate**
- **Spring Security**
